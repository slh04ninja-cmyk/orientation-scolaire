# 🎓 Système d'Orientation Scolaire

Application Streamlit qui classe automatiquement les élèves en trois
filières à partir de leurs notes :

| Code | Filière | Matières dominantes |
|------|---------|---------------------|
| **SM** | Sciences Mathématiques | Math + Physique |
| **SE** | Sciences Expérimentales | SVT + Physique |
| **LT** | Lettres & Traduction | Français |

## Démarrage rapide

```bash
pip install -r requirements.txt
streamlit run app.py
